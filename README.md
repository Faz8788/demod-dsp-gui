## 🚀 What this app does

demod-dsp-gui is a Windows app for viewing and managing DSP effects in a clean, game-like screen. It uses a framebuffer-style interface, so the display stays fast and easy to read. It is built for audio work, guitar effects, and Faust-based DSP tools.

Use it to:

- Load and view DSP setups
- Manage input sources in one place
- Switch between different screen sizes
- Handle effect chains with less clutter
- Keep your audio controls in a single window

## 🎛️ Main features

### 🖥️ Framebuffer-style GUI
The app uses a compact screen layout that keeps the interface easy to scan. It works well for live use and quick setup changes.

### 🎧 Multi-input handling
You can work with more than one input source. This helps when you switch between instruments, audio paths, or test signals.

## 🛠️ If something looks wrong

If the window does not open:

1. Check that you ran the correct file
2. Try right-clicking the app and choose Run as administrator
3. Make sure the ZIP file is fully extracted
4. Check that your Windows version is up to date

If audio does not work:

1. Check your audio cable
2. Pick the right input device in Windows
3. Make sure the app uses the same sample rate as your device
4. Close other apps that may use the same input

If the screen looks too small:

1. Resize the window
2. Maximize the app
3. Check your display scale in Windows settings
